# Запуск на Windows

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```cmd
npm install
```

## Запуск

### Для разработки
```cmd
npm run dev
```

### Для продакшена
```cmd
npm run build
npm run start
```

## Доступ к приложению

После запуска приложение будет доступно по адресам:
- http://localhost:5000 - локальный доступ
- http://YOUR_IP:5000 - доступ с других устройств в сети

## Настройка файрвола Windows

Чтобы разрешить доступ с других устройств:
1. Откройте "Параметры Windows" → "Обновление и безопасность" → "Безопасность Windows"
2. Перейдите в "Брандмауэр и защита сети"
3. Нажмите "Разрешить работу с приложением через брандмауэр"
4. Добавьте Node.js или разрешите порт 5000

## Переменные окружения

Создайте файл `.env` в корне проекта:
```
PORT=5000
HOST=0.0.0.0
NODE_ENV=development
```
